#画像の変え方

①画像変える
GF > settings > images 
フォルダ内に、画像を入れてください。（名前は自由）


②settings.jsonを書き換える
GF > settings > settings.json
をメモ帳などのエディタで開きます。

そしたら、
[
  {
    "ImageName": [
      "01.jpg",
      "02.jpg",
      "03.jpg",
      "04.jpg",
      "05.jpg",
      "06.png",
      "07.jpg",
      "08.jpg",
      "09.jpg",
      "10.jpg",
      "11.jpg"
    ]
  }
]

こんな感じで出てくると思うので
      "11.jpg"
の後ろにカンマ
次の行に追加したい画像名を入れます。
入れた結果として、
[
  {
    "ImageName": [
      "01.jpg",
      "02.jpg",
      "03.jpg",
      "04.jpg",
      "05.jpg",
      "06.png",
      "07.jpg",
      "08.jpg",
      "09.jpg",
      "10.jpg",
      "11.jpg",
	"img_sukinayatu.jpg"
    ]
  }
]

のようにしてください。

※※※※※※※※※※※※※※※※※※※※※※※※※※※※※※※※※※

なお注意点として、最後の画像名の後ろにはカンマを付けないでください。

↑多分settings.json変えるだけでいけるはず（22:33 2022/10/18）


③exeファイル化

Visual Studioの場合、メニュー上部のビルドを押します。
出てきたメニューの中で、ソリューションのビルドを押します。

その後、
GF > bin > Debug > GF.exe

が生成されています。（更新日時が、今の時間になっているかを必ず確認してください。）



④おまけ

exeファイル作成後、右クリックして、ショートカットの作成を押すと、exeファイルのショートカットが作れるので、
それを自分のＰＣの好きな場所に配置すると幸せになるかもしれない。

なお、ショートカットを作成したら、プラグラムが保管されているフォルダの位置をずらさないこと。
